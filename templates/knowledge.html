Platform Intelligence

Every field a trader fills in by memory — channel KPIs, bid ranges, vertical objectives, platform technical settings — is encoded here. Campaign Builder applies this knowledge automatically, in a defined order of precedence, so every campaign starts from the same expert baseline.

How defaults are applied

Rules stack in layers. More specific rules win. A field set by LOB × Channel overrides the same field set by Channel alone, which overrides Global.

Layer 1
Platform Defaults
TTD technical fields. Never in source documents. Always applied first.
Layer 2
Global
Applies to every campaign regardless of channel or vertical.
Layer 3
By Channel
KPIs, bids, and marketplace differ by channel (CTV vs Display vs Audio…).
Layer 4
By LOB
Objectives and goals differ by vertical (Auto vs Retail vs Healthcare…).
Layer 5 — wins
LOB × Channel
Most specific. Expert-level overrides for a known vertical + channel combination.

Any value from a source document (media plan, brief, etc.) always takes priority over all layers above.

Channel normalization — plain language → platform format

Source documents use inconsistent channel names. The tool recognizes all known variations and maps them to the correct TTD canonical value.

"Video CTV" / "Connected TV" / "Streaming TV" / "CTV"→CTV
"Video OLV" / "OLV" / "Online Video" / "Pre-roll"→OLV
"Display" / "Banner"→Display
"Native"→Native
"Streaming Audio" / "Audio Podcasts" / "Audio" / "Connected Car" / "Connected Home"→Audio
"DOOH" / "Out of Home" / "Digital OOH"→DOOH
